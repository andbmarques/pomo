A Simple Pomodoro Timer Using React Hooks


## Features

- Configurable Pomodoro Timer
- Light and Dark theme.


## Stack used

**Front-end:** 
- React
- React Timer Hook
- ChakraUI
- Phosphor Icons
- React Router DOM v6



## Running locally

Clone the project

```bash
  git clone https://github.com/andbmarques/pomo.git
```

Enter the project directory

```bash
  cd pomo
```

Install dependencies

```bash
  npm install
```
or
```bash
  yarn
```

Running

```bash
  npm run start
```
or
```bash
  yarn start
```
